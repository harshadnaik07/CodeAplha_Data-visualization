# CodeAplha_Data-visualization

This project analyzed supermarket sales data using Python.

Data visualization techniques such as bar charts,
line charts, scatter plots, histograms,
box plots, pie charts, and heatmaps
were used to identify business trends.

The analysis showed that product category,
payment method, city, and customer type
have a significant impact on overall sales.

These insights can help supermarket management
improve marketing strategies, inventory planning,
and customer satisfaction.
